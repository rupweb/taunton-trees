A more detailed description of these stately deciduous trees


Tilia euchlora - Lime

Description

A medium-sized tree with dark green leaves and arching branches. An elegant young tree, which becomes dense when mature and bears small fragrant creamy flowers in midsummer. It is a clean lime whose branches are free from aphids, but the flowers tend to have a narcotic effect apon bees.

Suggested Uses

Formal stately garden tree

Size

From 10 feet high
